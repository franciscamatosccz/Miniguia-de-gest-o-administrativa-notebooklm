# Miniguia-de-gest-o-administrativa-notebooklm

## Contexto

Este mini guia foi desenvolvido com o intuito de reunir e organizar conhecimentos sobre Gestão de Processos Administrativos em um único ambiente de estudo utilizando o NotebookLM. 

O conteúdo foi estruturado a partir de fontes confiáveis para facilitar consultas, aprofundar conceitos e compreender as principais práticas relacionadas ao planejamento, mapeamento, padronização e melhoria de processos nas organizações.


## Objetivos

Aprofundar os meus conhecimentos em um assunto que possuo interesse, no intuito de possuir um suporte em meus estudos e por meio disso me aprimorar enquanto profissional.

Compreender os fundamentos da gestão de processos administrativos.

Estudar métodos de mapeamento, padronização e melhoria contínua de processos.

Conhecer indicadores utilizados para avaliar a eficiência dos processos administrativos.

Reunir informações sobre gestão documental e ferramentas de apoio à administração.

Criar uma base de conhecimento organizada que facilite consultas, revisões e o aprendizado contínuo sobre o tema.
